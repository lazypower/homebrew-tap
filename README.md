# Homebrew Tap

Homebrew formulae and casks for [lazypower](https://github.com/lazypower) projects.

## Usage

```bash
brew tap lazypower/tap
```

### Continuity

Persistent memory for AI coding agents. Single binary, zero dependencies.

```bash
brew install lazypower/tap/continuity
```

### VoidReader

Native macOS markdown viewer with a reader-first philosophy.

```bash
brew install --cask lazypower/tap/voidreader
```
